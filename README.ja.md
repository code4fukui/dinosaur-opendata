# 恐竜オープンデータ

このプロジェクトは、福井県立恐竜博物館の予約状況データと福井で発見された恐竜のオープンデータを提供するものです。

## デモ
[恐竜博物館ダッシュボード](https://code4fukui.github.io/dinosaur-opendata/)では、福井県立恐竜博物館の60日先までの予約状況を確認できます。

## 機能
- 福井県立恐竜博物館の予約状況(人数、金額、平均単価)を確認できる
- 福井で発見された恐竜の情報を閲覧できる

## データ・API
- [福井県立恐竜博物館の予約状況CSV](https://code4fukui.github.io/dinosaur-opendata/latest_dino_sum.csv)
- [福井で発見された恐竜のオープンデータ](https://code4fukui.github.io/dinosaur-opendata/dinosaur-fukui.csv)
