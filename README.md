# 恐竜博物館ダッシュボード

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Open data repository. Columns: name, name_sci, name_sci_ja, name_alt.

## Demo

https://code4fukui.github.io/dinosaur-opendata/

## Features

- ES module compatible with Deno and modern browsers
- Open data CSV with columns: name, name_sci, name_sci_ja, name_alt

## Data

Columns: name, name_sci, name_sci_ja, name_alt, year_excavation, year_named

## Project Structure

- `data/` — Data files

## Installation & Setup

No build step required. Clone and open in a browser or use with Deno:

```bash
git clone https://github.com/code4fukui/dinosaur-opendata.git
```
